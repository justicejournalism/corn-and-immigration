# “Corn & Immigration”

![image](http://justicejournalism.org/wp-content/uploads/2015/03/corn-project-e1427223629269.png)

What does corn have to do with the mass movement of millions people from Mexico to the United States? An interactive timeline and map shows how agriculture trends and trade policies correlate with spikes in immigration. Team members told a lot of “corny” jokes while producing this project.
